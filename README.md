## Diferença entre VM e VPS e insfraestrutura em nuvem
VM local: Maquina virtual que corre no meu próprio computador.
VPS: Servidor virtual alugado num datacenter, acessado pela internet, com recurso dedicados ou partilhados.
Infraestrutura em nuvem: Conjunto de serviços oferecidos por provedores de cloud, permitindo criar e escalar vários servidores conforme a necessidade.
